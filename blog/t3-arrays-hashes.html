Arrays and Hashes

A Bit About Arrays

Arrays are lists of things.

  • They are ordered
  • They have an index
    • an index is an integer value associated with the postion of an element in an array.
    • Array indexes start counting with zero.
      • Hence the first item in an array would be item 0
      • The last item in an array of 7 elements would be Item 6

Hashes

Hashes are a list of key-word pairs.

  • They are not ordered, not really.. Lemme explain:
    • Technically the first item in a hash is the first item entered into a hash.
      • But you could never, for example, jump to item Six or item One in a hash. There is no item six, and the sixth item entered into the hash is not labeled as such in any meaningful way.
  • Key-word pairs are two words or phrases you choose to asssociate with each other.
  • "Cars: Ford" is an example of a key-word pair.
  • There are no rules regarding pairing. They can also be non-sensical
    • Flightless-Bird: Stalagmite
  • To call an element in a hash you call its key-word.
    • Calling key-word "Cars" will yeild element "Ford". Calling key-word "Flightless-Bird" yeilds ""Stalagmite"".
