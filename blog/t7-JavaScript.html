It might be a choice of vocabulary.

In java an object refers to a collection of properties, that we can add to remove from as we please.

var day1 = {
  squirrel: false,
  events: ["work", "touched tree", "pizza", "running",
           "television"]
};

What you see above is an object called "day1". It is defined in "Object Literal Notation".
"Object Literal Notation" means that we created the object and its explicit properties (squirrel and events) all at one time.
The value of "Squirrel" is false, and the value of "events" is an array.
